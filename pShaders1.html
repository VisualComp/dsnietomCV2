Image Processing with Pixels

Load Images pixel by pixel

March 26, 2020 David Nieto

Poniendo en practica los conceptos aprendidos en el estudio de matrices de pixeles que se vio en el ejemplo anterior PAINTING BACKGROUNDS WITH PIXELS y la documentación del sitio web processing.org, podremos ahora empezar a trabajar de la misma manera con imágenes.

En este caso podemos ver la imagen como una cuadricula compuesta por pixeles, donde cada pixel está compuesto por cuatro piezas de información (R, G, B, A), esto quiere decir que, como se vio en el ejemplo anterior, es posible acceder al valor de cada pixel y modificarlo de tal manera que podamos cambiar o modificar ciertas características especificas de la imagen. Para ello se usará el mismo proceso de búsqueda de índices x + y * width * 4 para recorrer la matriz de pixeles de la imagen y acceder a los valores RGBA de cada pixel.

El siguiente programa tiene el propósito de ilustrar la forma en la que se cargan imágenes a un lienzo, pero en lugar de mostrarla se pintara cada pixel del lienzo según los valores y posición de cada pixel de la imagen. Para ello tanto el lienzo como la imagen deben ser del mismo tamaño, o por lo menos el mismo ancho, con el fin de que la posición del pixel de la imagen corresponda a la posición del lienzo (mas adelante se verá como pintar un lienzo con dimensiones distintas al de la imagen). Adicionalmente se podrá interactuar con la ventana por medio de Sliders para modificar los valores RGB de los pixeles.

Image of Agalychnis callidryas, known as the Red Eyed Treefrog. Source: Wikipedia, the free encyclopedia.

Red-eyed Tree Frog (Agalychnis callidryas), photographed near Playa Jaco in Costa Rica

p5*js Code

let rSlider, gSlider, bSlider; //Declarar variables de sliders
let img; // Declarar variable 'img'

function setup() {
  createCanvas(800, 600);
  textSize(14);
  noStroke();

  // Crear sliders
  rSlider = createSlider(-255, 255, 0);
  rSlider.position(20, 20);
  gSlider = createSlider(-255, 255, 0);
  gSlider.position(20, 50);
  bSlider = createSlider(-255, 255, 0);
  bSlider.position(20, 80);
  img = loadImage('https://upload.wikimedia.org/wikipedia/commons/thumb/b/be/Red_eyed_tree_frog_edit2.jpg/800px-Red_eyed_tree_frog_edit2.jpg'); 
}

function draw() {
  const R = rSlider.value();
  const G = gSlider.value();
  const B = bSlider.value();
  loadPixels();
  img.loadPixels(); //Carga los pixeles de la imagen
  // Recorre la matriz de pixeles
  for (let y = 0; y < height; y++) {
		for (let x = 0; x < width; x++){ 
			let index = (x+y*width)*4; // Posicion del pixel
			let r=img.pixels[index+0]; // Componente Red
			let g=img.pixels[index+1]; // Componente Green
			let b=img.pixels[index+2]; // Componente Blue
			let a=img.pixels[index+3]; // Componente Alpha
			// Pinta el lienzo con los valores de los pixeles de la imagen
			// Suma el valor del componente junto con el valor del slider
			pixels[index+0]=r+R; 
			pixels[index+1]=g+G; 
			pixels[index+2]=b+B;
			pixels[index+3]=a;
		}
	}
  updatePixels();	
  
  stroke(255);
  textStyle(BOLDITALIC);
  text('RED', 175, 40);
  text('GREEN', 175, 70);
  text('BLUE', 175, 100);
}
