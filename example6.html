Processing transition - p5*js Examples

March 19, 2020 David Nieto

Los siguientes ejemplos tienen el propósito de desarrollar y practicar la transición entre el lenguaje Processing y el lenguaje p5.js observando sus principales diferencias, ya que ambos lenguajes se ven muy similar, y cómo convertir de uno a otro.

Este programa está basado en un sketch básico de ejemplo, encontrado en el sitio web processing.org, en el lenguaje Processing, el cual tiene como propósito cargar y mostrar imágenes dentro del canvas. Crea una ventana con un ancho de 1024px y una altura de 512px y luego muestra una imagen, cargada previamente, repitiéndola varias veces y distribuyendo cada copia por todo el lienzo, modificando sus propiedades como el tamaño, posición y orientación, formando un collage.

Image of Lena Forsén ("Lenna") used in many image processing experiments. Source: Wikipedia, the free encyclopedia.

p5*js Code

let img; // Declarar variable 'img'
function setup() { 
	createCanvas(1024, 512);
	background(210);
	img = loadImage('https://upload.wikimedia.org/wikipedia/en/7/7d/Lenna_%28test_image%29.png'); // Cargar la imagen
} 

function draw() {
	imageMode(CENTER);
	// Muestra la imagen en su tamaño original en la posición en el centro del canvas
	image(img, width / 2, img.height / 2);

	imageMode(CORNER);
	// Muestra la imagen en la posición (0,0) del canvas a la mitad del tamaño
	image(img, 0, 0, img.width / 2, img.height / 2);

	imageMode(CORNERS);
	// Muestra la imagen con su esquina inferior derecha en la posición (width, height)
	image(img, width/2+img.height/2, height/2, width, height);

	imageMode(CORNER);
	// Muestra la imagen pegada a la derecha de la imagen de tamaño real, a la mitad del tamaño
	image(img, width/2-img.height, height/2, img.width / 2, img.height / 2);
	
	imageMode(CORNER);
	// Muestra la imagen en la posición (0,0) con respecto a la imagen de tamaño real, a 1/4 del tamaño
	image(img, width/2-img.height/2, 0, img.width / 4, img.width / 4);
}
