Processing transition - p5*js Examples

March 12, 2020 David Nieto

Los siguientes ejemplos tienen el propósito de desarrollar y practicar la transición entre el lenguaje Processing y el lenguaje p5.js observando sus principales diferencias, ya que ambos lenguajes se ven muy similar, y cómo convertir de uno a otro.

Este programa está basado en un sketch básico encontrado en el sitio web p5js.org, en el lenguaje p5.js, con el fin de observar un ejemplo básico que sirva como transición para usuarios de Processing que están empezando en p5.js y así realizar su primer bosquejo. Crea una ventana con un ancho de 600px y una altura de 400px, y luego empieza a dibujar elipses rosas de 50px por 50px en la posición donde se encuentre el mouse . Haciendo uso de la función 'mouseIsPressed', cuando se presione el botón del mouse las elipses que se dibujen cambiaran de color a negro, dando la ilusión o posibilidad de dibujar a voluntad con el mouse.

p5*js Code

function setup() { 
	createCanvas(600, 400);
	background(210);
} 

function draw() {	
	// Cambia el color de las elipses mientras este presionado el boton del mouse
	if (mouseIsPressed) {
		fill(0);
	} else {
		fill(255,0,128);
	}
	// Dibuja una elipse en la posicion del mouse
	ellipse(mouseX, mouseY, 50, 50);
}
