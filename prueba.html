Basic Sketch

Processing transition - p5*js Examples

March 12, 2020 David Nieto

Este programa esta basado en un sketch básico. Crea una ventana con un ancho de 600 pixeles y una altura de 400 pixeles, y luego empieza a dibujar un cuadrado azul según la posición del mouse, al mismo tiempo el fondo de la ventana se va rellenando de color gris, dando la ilusión que solo existe un cuadrado azul moviéndose por toda la ventana siguiendo al mouse.

Image of Blue Peafowl. Indian peacock displaying its tail. Source: Wikipedia, the free encyclopedia.

A peacock showing off its colours, Sultanpur National Park

Code

function setup() { 
	createCanvas(600, 400);
} 

function draw(){ 
	// Dibuja un nuevo background en cada frame
	background(210);
	stroke(0);
	fill(0,0,255);
	// En modo CENTER, el rectángulo se dibuja en el centro del cursor
	rectMode(CENTER);
	// mouseX y mouseY posición del ratón
	rect(mouseX,mouseY,50,50);
}
